## About

### Dependencies
```bash
brew install nvim fzf ripgrep
gem install solargraph
```

### Installing

```bash
git clone --recurse-submodules https://github.com/jairovm/vim.git ~/.vim
ln -nsf ~/.vim ~/.config/nvim
ln -nsf ~/.vim/init.vim ~/.vimrc
```
